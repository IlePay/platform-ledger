# Platform Ledger API Documentation

## Base URL
```
Production: https://api.platform.com/v1
Development: http://localhost:8082/api/v1
```

## Authentication
Currently no authentication (will be added in v0.2.0)

## Rate Limiting
- 100 requests/minute per IP
- 1000 requests/hour per IP

## Error Codes
- 200: Success
- 201: Created
- 400: Bad Request
- 404: Not Found
- 500: Internal Server Error

## Endpoints

See interactive documentation at `/swagger/index.html`
